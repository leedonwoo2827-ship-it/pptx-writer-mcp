# /convert-to-pptx

마크다운 파일을 읽고 개조식으로 변환하여 PPTX 프레젠테이션을 생성하는 워크플로우.

## 실행 흐름

### Step 1: 세션 시작 (create_pptx)

`create_pptx` 도구로 PPTX 세션을 시작한다.

- 사용자에게 출력 파일명과 project_dir을 확인
- 회사 템플릿이 있으면 template_file 지정
- 반환된 session_id를 기억

### Step 2: 마크다운 구조 파악

마크다운 파일을 읽어 전체 구조(H1/H2 섹션)를 파악한다.

- 파일이 크면 처음 200줄만 먼저 읽어 목차 구조 확인
- 총 섹션 수와 예상 슬라이드 수를 사용자에게 안내

### Step 3: 섹션별 변환 (반복)

각 섹션을 읽고 슬라이드 마크다운으로 변환한다.

- 줄글 → 개조식(불릿 포인트)으로 변환
- 표는 그대로 유지
- 이미지 참조도 그대로 유지
- `add_slides` 도구로 슬라이드 추가
- 모든 섹션이 끝날 때까지 반복

개조식 변환 원칙:
- 서술형 문장을 핵심 키워드 + 수치 중심으로 압축
- 한 불릿에 1~2줄 이내
- 슬라이드당 불릿 5~7개 이내

### Step 4: 저장 (finalize_pptx)

`finalize_pptx` 도구로 파일을 저장하고 결과를 안내한다.
